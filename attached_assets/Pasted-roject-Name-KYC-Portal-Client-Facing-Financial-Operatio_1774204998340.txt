✨ ADVANCED ANIMATION SPECIFICATIONS
Page Load Sequence (staggered orchestration)
0ms    — Background mesh gradients fade in (opacity 0→1, 400ms)
100ms  — Sidebar slides in from left (translateX -72px → 0, spring)
150ms  — TopBar slides down from top (translateY -64px → 0, spring)
250ms  — Page title fades up (translateY 20px → 0, 400ms ease-out)
350ms  — KPI cards enter one by one (translateY 24px → 0, stagger 80ms each)
550ms  — Chart widgets fade + scale (scale 0.97→1, opacity 0→1, 350ms)
700ms  — Right panel slides in from right (translateX 40px → 0, spring)
900ms  — Transaction rows appear with stagger (opacity 0→1, translateY 8px→0)
1100ms — Ring chart animations begin (stroke-dashoffset draws)
Micro-interactions

Number counters: KPI numbers count up from 0 on mount using a custom counter hook, easeOutCubic, 1200ms duration
Ring draws: SVG stroke-dashoffset animated with cubic-bezier(0.34, 1.56, 0.64, 1) (slight overshoot)
Card hover: translateY(-4px) scale(1.01) + deepened shadow, transition: all 200ms cubic-bezier(0.34, 1.56, 0.64, 1)
Button press: scale(0.96) on mousedown, releases on mouseup
Pill tab switch: shared layout animation, lime background morphs between pills (no hard cut)
Notification badge: appears with scale(0→1) spring + subtle rotate(12deg→0) shake
Popover open: scales from 0.95, opacity 0 → 1.0, opacity 1, origin at trigger element, spring(stiffness: 400, damping: 30)
Route transition: outgoing page fades + slides up 12px, incoming page fades in + slides up from 12px below, 250ms overlap
Tooltip on bar chart: follows cursor smoothly with 60ms lag (lerp), doesn't teleport
Donut hover: hovered segment moves outward 6px with cubic-bezier(0.34, 1.56, 0.64, 1) spring bounce
Card 3D tilt: smooth requestAnimationFrame tracking, max ±8° on any axis, perspective: 800px
Transaction row hover: left accent bar slides in from scaleX(0→1), transform-origin: left, 150ms

Ambient Animations (always running, low CPU)

Background grain: subtle @keyframes grain-shift — transform: translate(0,0) to translate(4px, 4px) on 8s loop
Active nav item: lime accent border pulses opacity 0.7→1.0 on 3s loop
Live status dot: @keyframes pulse-ring — concentric ring expands from dot, fades out, 2s loop
Logo arcs: counter-rotate slowly, animation: spin-cw 20s linear infinite and spin-ccw 28s linear infinite
